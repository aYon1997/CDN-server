Stagewise 插件安装指南

一键自动安装或手动配置 - 选择最适合您的方式

自动安装 Stagewise

推荐方式：一键自动安装

最快捷的安装方式，适用于大多数开发者

1

在扩展市场中安装 Stagewise

打开 Cursor 编辑器，进入扩展市场，搜索 "stagewise" 并安装

2

激活 Stagewise 自动设置

使用快捷键 Ctrl + Shift + J 或在命令面板中搜索 "Stagewise Auto-setup"

3

自动配置过程

Stagewise 会自动在对话框中输入提示词，识别您的前端技术栈

4

安装依赖

根据识别结果，自动安装所需的依赖包到项目中

点击 stagewise 后会自动在 cursor 对话框输入提示词

自动识别技术栈

智能检测项目中的前端框架和技术栈，自动配置对应环境

零配置集成

无需手动修改配置文件，所有配置自动完成

快速回滚

如不满意自动配置，可一键恢复到原始状态

手动安装 Stagewise

高级方式：手动配置

完全控制安装过程，适用于自定义配置需求

1

安装依赖

在 package.json 中添加必要的开发依赖

{ 
  "devDependencies": { 
    "@babel/plugin-proposal-nullish-coalescing-operator": "^7.18.6", 
    "@babel/plugin-proposal-optional-chaining": "^7.21.0", 
    "@babel/preset-env": "^7.27.2", 
    "@stagewise/toolbar": "^0.2.1" 
  } 
}
2

配置 Babel

在 babel.config.js 中添加必要的预设和插件

module.exports = {
  presets: [['@vue/cli-plugin-babel/preset', { 
    useBuiltIns: 'usage', 
    corejs: 3,
    targets: { browsers: ['> 1%', 'last 2 versions', 'not dead'] }
  }]],
  env: {
    development: {
      plugins: [
        'dynamic-import-node',
        '@babel/plugin-proposal-nullish-coalescing-operator',
        '@babel/plugin-proposal-optional-chaining'
      ]
    }
  }
}
3

初始化工具栏

在 main.js 中初始化 Stagewise 工具栏

import Vue from 'vue'
import { initToolbar } from '@stagewise/toolbar'

if (process.env.NODE_ENV === 'development') {
  initToolbar()
}

new Vue({
  el: '#app',
  router,
  store,
  render: h => h(App)
})
4

转译依赖

在 vue.config.js 中配置转译

module.exports = {
  transpileDependencies: ['@stagewise/toolbar']
}

完全控制

精确控制每个配置细节，满足特殊项目需求

复杂项目支持

适用于有特殊配置要求或自定义构建流程的项目

学习价值

深入了解工具集成原理，提升技术能力
